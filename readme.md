# 構想

疾患はゲノム構造の影響を受けた下流の遺伝子が引き起こす生体メカニズムの異常と定義する。上流（ゲノム）から下流（疾患メカニズム）の間で中心を担う遺伝子を見つけることで創薬標的探索を見出す

## 考え方

1. 特定の疾患のSNPsを調査し、SNPに関連のある遺伝子（SNP関連遺伝子）を特定する
2. SNP関連遺伝子と繋がりのある遺伝子群（PPI遺伝子群）を調べる
   1. Signore、Reactome、String、BioGRIDを選べるようにする（デフォルトはSignore, Reactome、String（Score>0.7））
   2. 繋がりは１階層から３階層まで選べるようにする（デフォルトは１階層）
3. SNP関連遺伝子とPPI遺伝子群のネットワークの中心性解析を行う
   1. SNP関連遺伝子からのフローと考えて、最初の流量を下記のように定義する
      1. GWAS P値（-log10)
      2. LoF、GoF、Misssenseなどvariantと遺伝子機能への影響度を数値化
   2. SNP関連遺伝子からのフローをベースにネットワークの流れを計算し、重要遺伝子をスコア化する
4. 重要遺伝子群を用いたエンリッチメント解析をローカルで実施
   1. Reactome.GeneOntology,HPO
   2. エンリッチメント解析結果を図で示す
5. SNP-SNP関連遺伝子ーPPI重要遺伝子のサンキーダイアグラム、遺伝子ネットワーク図（Plotly）を作成
6. エンリッチメント解析の重要パスウエイごとに重要遺伝子をリストアップして示す
